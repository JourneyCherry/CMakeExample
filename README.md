# How To Use

1. In Linux, make 'build' folder in it and run cmake


'''
mkdir build
cd build
cmake ..
'''

2. build & install program and run it.

'''
make all
make install
../Bin/pluser
'''

# 사용법

1. 리눅스에서, 'build' 폴더를 만들고 cmake를 실행하세요.

2. 프로그램을 빌드 및 설치하고 실행하십시오.
